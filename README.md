# Next.js with TypeScript Template

Shopify の js-buy-sdk を使用して作った EC フロント。
Next.js を使用しています。
https://shopify-store-front-with-nextjs-jjx3prsx7.vercel.app/

## how to use

### 開発時

```
yarn dev
```

### test

```
yarn test
```

### lint

```
yarn lint
```

### その他

pre-commit で型チェック・lint・test 行うようになっています。
